1.  Backend:
    goto cd Backend
    Update .env with MONGO_URI and JWT_SECRET.
    npm install express mongoose bcryptjs jsonwebtoken moment cors dotenv
 

2.  Return to main directory

3.  Frontend:
    goto cd Frontend
    npm install react react-dom react-router-dom react-bootstrap axios react-toastify react-tsparticles @mui/icons-material react-datepicker moment.
    

4.  Return to main directory and npm install

6.  npm start
